**Step 3: Data Migration (1-2 hours)**
- Sync all historical orders (with cutoff date)
- Import product catalog
- Import retailer/customer database
- Set up banking integration

**Step 4: Training Session (1 hour)**
- Walk through portal sections
- Demonstrate AI employees
- Set up automations
- Configure team members
- Create first tasks and calendar events

**Step 5: Go-Live Checklist**
- [ ] All stores synced
- [ ] AI employees responding
- [ ] Cron jobs active
- [ ] Team members added
- [ ] Client can log in and navigate
- [ ] First daily report generated

### SOP 2: AI Employee Configuration

**Per Client:**
1. Define 5 AI employee personas based on client's business
2. Write system prompts with client-specific data points
3. Generate headshot photos via Gemini
4. Configure skills, tools, connectors per employee
5. Seed initial conversations for warmup
6. Test responses for accuracy and tone
7. Monitor first 50 conversations for quality

### SOP 3: Monthly Client Review

1. Generate monthly report via `/api/reports/generate`
2. Review: revenue trends, order health, retailer retention
3. Identify top 3 action items
4. AI employee generates summary + recommendations
5. Share report with client (PDF + portal link)
6. Schedule follow-up call if needed

### SOP 4: New Integration Deployment

1. Research API documentation
2. Create lib file (`src/lib/{service}-api.ts`)
3. Create API routes (`src/app/api/{service}/`)
4. Add to automations table
5. Create cron job for sync
6. Test with sandbox/staging data
7. Deploy to production
8. Monitor first 48 hours

### SOP 5: Support Escalation

| Level | Response Time | Who Handles |
|-------|--------------|-------------|
| L1: UI questions | < 4 hours | AI employee (auto) |
| L2: Data issues | < 12 hours | Support team |
